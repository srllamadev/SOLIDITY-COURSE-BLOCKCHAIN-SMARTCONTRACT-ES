# Primeros Pasos - Desarrollo Web Básico

Esta carpeta contiene los fundamentos básicos de desarrollo web (HTML y CSS) antes de adentrarse en Solidity. Es el punto de partida para crear interfaces que interactúen con contratos inteligentes.

## Archivos en esta carpeta

### doc.txt
Documento que describe los requisitos previos para el curso:
- Tener un IDE (como Visual Studio Code)
- Conocimientos básicos de HTML, CSS y JavaScript
- Estructura del aprendizaje: primero web básico, luego Solidity

### index.html
Página HTML que simula la interfaz de un "Banco Descentralizado". Incluye:
- Estructura básica de HTML5
- Header con título y descripción
- Secciones principales con tarjetas (cards)
- Formulario de contacto
- Botones de acción
- Footer con copyright

Esta página sirve como maqueta de cómo podría verse una dApp (aplicación descentralizada) en la web.

### estilos.css
Hoja de estilos CSS que da formato a la página. Características destacadas:
- Variables CSS para paleta de colores (tema rosa)
- Reset de estilos para consistencia
- Diseño responsivo con grid
- Estilos para tarjetas, formularios y botones
- Transiciones suaves

## Propósito

Esta carpeta establece las bases del desarrollo frontend antes de integrar contratos inteligentes. Una dApp típica combina:
1. **Frontend**: HTML/CSS/JS para la interfaz de usuario
2. **Backend**: Contratos inteligentes en Solidity
3. **Conexión**: Web3.js o similar para interactuar con la blockchain

## Cómo usar

1. Abre `index.html` en un navegador para ver la página
2. Modifica `estilos.css` para cambiar colores o layout
3. Usa esta estructura como base para futuras interfaces de dApps

Después de dominar estos conceptos básicos, estarás listo para conectar la interfaz con contratos Solidity reales.
